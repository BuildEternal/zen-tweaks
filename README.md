## Overview

**Zen Tweaks** comes with the following configurable features:

- Fix Misaligned Split Mode Dragger
- Remove Gaps
  - Remove Tab Rounded Corners
  - Remove Dragger Gap in Split Mode
- Remove Tab Highlight in Split Mode
- Remove Tab Shadow
